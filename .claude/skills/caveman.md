# Caveman Skill

## Descripción

Caveman es una skill para agentes que permite explicar conceptos técnicos de manera simple, usando analogías y lenguaje sencillo, como si se lo explicaras a un "cavernícola". Es útil para aclarar temas complejos a usuarios sin conocimientos técnicos.

## Uso

- Cuando el usuario solicite una explicación "como para cavernícola" o "como para un niño".
- Cuando el usuario pida simplificar un concepto técnico.
- Cuando el usuario use palabras clave como: simple, fácil, básico, analogía, cavernícola, niño, explicación sencilla.

## Ejemplo de prompt

> "Explícame qué es una API como si fuera un cavernícola."

**Respuesta esperada:**

> "Imagina que tienes una piedra mágica (API) que, cuando la golpeas de cierta forma (haces una petición), te da comida o fuego (datos o servicios). No necesitas saber cómo funciona por dentro, solo cómo pedirle lo que quieres."

## Instrucciones para el agente

- Usa frases cortas y palabras simples.
- Utiliza analogías cotidianas o primitivas.
- Evita tecnicismos y explicaciones largas.
- Si el usuario pide más detalle, aumenta la complejidad gradualmente.
